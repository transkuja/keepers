Lorsqu'on laisse le prisonnier seul, un message apparaît, mais ce message apparaît à chaque fois qu'on change de tile.
(il ne devrait s'afficher qu'un fois lorsqu'on quitte LA tile sur laquelle se trouve le prisonnier)

Le joueur est obligé d'utiliser tous ses points d'action pour passer un tour

On peut trade avec un joueur sur un tile distant.

Les pions bougent lorsqu'ils sont collés.

Bug d'inventaire: Lorsqu'on ramasse une baie avec quelque chose dans l'inventaire: elle disparaît.
		  Lorsqu'on ramasse une baie sans rien dans l'inventaire, elle est ramassée, mais tous les autres objets ramassés ensuite disparaissent.
